# go-experimental

## About
go-experimental is a collection of experiments with Go. It is published on <https://github.com/vbsw/go-experimental>.

**a01** - conditional compilation  
**a02** - nested interfaces (expected behaviour: does not compile)  

**b01** - binary search benchmark  
**b02** - calls from Go into C and from C into Go benchmark (needs Cgo)  
**b03** - simple wildcard string matching benchmark (needs Cgo)  
**b04** - benchmark of github.com/vbsw/go-lib/tabformat (needs Cgo)
